Introduction

Modules 7 and 8 focused on the Mandelbrot Set and the Julia Set. The Mandelbrot Set is a fractal defined in the complex plane, known for its intricate and self-repeating structures. It is generated by iterating the function \( z = z^2 + c \), where \( z \) and \( c \) are complex numbers [1]. M. The Mandelbrot set is named after mathematician Benoit Mandelbrot, who studied it in 1980.

First Code: Constructing the Matrix

The first code construct the 2 by 2 matrix. 


import numpy as np
import matplotlib.pyplot as plt

# Create a simple 2D grid of points using NumPy
def generate_grid(size):
    x = np.linspace(-1, 1, size)  # Generate 'size' number of points between -1 and 1
    y = np.linspace(-1, 1, size)
    grid = np.meshgrid(x, y)  # Create a 2D grid of points
    return grid

# Visualize the grid of points
def plot_grid(grid):
    x_grid, y_grid = grid
    plt.figure(figsize=(10, 10))
    plt.scatter(x_grid, y_grid, s=1, color='blue')  # Plot the grid points
    plt.title(f'2D Grid of {len(x_grid)}x{len(y_grid)} points')
    plt.show()

# Set the size of the grid
size = 50  # You can adjust this to make the grid denser or sparser

# Generate and plot the grid
grid = generate_grid(size)
plot_grid(grid)

Second Code: Constructing the Julia Set (Iteration 1)

The second code transforms the 2 by 2 grid through a series of iterations. Each point in the grid is treated as a complex number and undergoes 1 iteration of the function \( z = z^2 - 0.5 \). The number of iterations is set to one, resulting in a slightly distorted version of the original grid. The final output is a scatter plot of the transformed grid.



def iterative_transform(grid, iterations):
    x_grid, y_grid = grid
    z = x_grid + 1j * y_grid  # Create a complex plane
    for i in range(iterations):
        z = z ** 2 - 0.5  # Iteratively apply the transformation
    return np.real(z), np.imag(z)

# Set the number of iterations
iterations = 1

# Apply the iterative transformation
fractal_grid = iterative_transform(grid, iterations)

# Plot the fractal grid
plot_grid(fractal_grid)
