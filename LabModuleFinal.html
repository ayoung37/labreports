Introduction

Modules 7 and 8 focused on the Mandelbrot Set and the Julia Set. The Mandelbrot Set is a fractal defined in the complex plane, known for its intricate and self-repeating structures. It is generated by iterating the function \( z = z^2 + c \), where \( z \) and \( c \) are complex numbers [1]. The Mandelbrot set is named after mathematician Benoit Mandelbrot, who studied it in 1980.

First Code: Generates the Mandelbrot Set

The first code calculates a portion of the Mandelbrot set using the Numba function. It zooms into a detailed area of the fractal and visualizes it with a color map that shows how quickly each point escapes.

The code begins by importing the following functions:
import numpy as np,
import matplotlib.pyplot as plt,

The function 'mandelbrot(c, max_iter)' calculates the number of iterations required for a complex number \( c \) to escape the boundary of the Mandelbrot set.
The function 'mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter)' generates a 2D array that represents the Mandelbrot set over a specified region of the complex plane.
The parameters 'xmin, xmax, ymin, ymax = -0.75, -0.74, 0.1, 0.1' define the zoomed-in area of the Mandelbrot set, while the code 'plt.imshow(n3.T, extent=[xmin, xmax, ymin, ymax], cmap='hot')' is used to plot the fractals. 


import numpy as np
import matplotlib.pyplot as plt
from numba import jit

# JIT-optimized mandelbrot function
@jit(nopython=True)
def mandelbrot(c, max_iter):
    z = 0
    n = 0
    while abs(z) <= 2 and n < max_iter:
        z = z*z + c
        n += 1
    return n

# JIT-optimized mandelbrot set generator
@jit(nopython=True)
def mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter):
    r1 = np.linspace(xmin, xmax, width)
    r2 = np.linspace(ymin, ymax, height)
    n3 = np.empty((width, height))

    for i in range(width):
        for j in range(height):
            n3[i, j] = mandelbrot(r1[i] + 1j * r2[j], max_iter)

    return n3

# Parameters for the plot
xmin, xmax, ymin, ymax = -2.0, 1.0, -1.5, 1.5
width, height = 800, 800
max_iter = 100

# Generate the Mandelbrot Set
n3 = mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter)
# Parameters for the initial zoom level
xmin, xmax, ymin, ymax = -0.75, -0.74, 0.1, 0.11  # Zoom into this specific area
width, height = 800, 800  # Resolution of the image
max_iter = 1000  # Number of iterations

# Generate the Mandelbrot Set with zoom

# Plotting the zoomed Mandelbrot Set
plt.figure(figsize=(10, 10))
plt.imshow(n3.T, extent=[xmin, xmax, ymin, ymax], cmap='hot')
plt.colorbar()
plt.title("Mandelbrot Set - Zoomed In")
plt.show()

Third Code: The Julia Set

The third code focus on the Julia Set. The Julia Set.
The code begins by importing the following functions:
import numpy as np
import matplotlib.pyplot as plt
from numba import jit
The function julia_set(c, xmin, xmax, ymin, ymax, width, height, max_iter) computes the Julia set for a specific complex number c. It iteratively calculates z = z*z + c for every point z in the complex plane. 


import numpy as np
import matplotlib.pyplot as plt
from numba import jit

# JIT-optimized function to generate the Julia Set
@jit(nopython=True)
def julia_set(c, xmin, xmax, ymin, ymax, width, height, max_iter):
    r1 = np.linspace(xmin, xmax, width)
    r2 = np.linspace(ymin, ymax, height)
    n3 = np.empty((width, height))

    for i in range(width):
        for j in range(height):
            z = r1[i] + 1j * r2[j]
            n = 0
            while abs(z) <= 2 and n < max_iter:
                z = z*z + c
                n += 1
            n3[i, j] = n

    return n3

# Parameters for the plot
xmin, xmax, ymin, ymax = -1.5, 1.5, -1.5, 1.5
width, height = 800, 800  # Image resolution
max_iter = 256  # Number of iterations for accuracy

# Choose a value for the complex constant c
c = complex(-0.8, 0.156)

# Generate the Julia Set
n3 = julia_set(c, xmin, xmax, ymin, ymax, width, height, max_iter)

# Plotting the Julia Set
plt.figure(figsize=(10, 10))
plt.imshow(n3.T, extent=[xmin, xmax, ymin, ymax], cmap='hot', interpolation='bilinear')
plt.colorbar()
plt.title(f"Julia Set for c = {c}")
plt.show()
